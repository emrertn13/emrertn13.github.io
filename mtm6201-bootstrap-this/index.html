Button Styling

Button Text Color

Default button text. To make the element a button we add the class .btn.

To change text color of the button into a blue we add the class .text-primary.

To change the text color & the outline of the button into a blue we add the class .outline-primary.

Button Background color

Default button background. To make the element a button we add the class .btn.

To change background color of the button into a blue we add the class .btn-bg-primary.

To change background area color of the button into a blue we add the class .alert-primary.
